for each new request you should structure your prompt like this for spot-on results!

1. GOAL

STATE PRECISELY WHAT YOU WANT.
CLEAR INTENT LEADS TO BETTER RESULTS.

2. RETURN FORMAT

SPECIFY THE OUTPUT STYLE, SUCH AS A LIST, TABLE, SCRIPT, OR OUTLINE.

3. WARNINGS

SET BOUNDARIES FOR TONE, ACCURACY, OR CONSTRAINTS.

4. CONTEXT

SHARE RELEVANT BACKGROUND, GOALS, OR AUDIENCE DETAILS. MORE CONTEXT MAKES THE RESPONSE FEEL THOUGHTFUL, NOT GENERIC.
